ボールを打ち返し、相手のゴールに叩き込め！\

## カーブショット
- ボールとプレイヤーが**接触した瞬間**に、上 or 下方向を入力すると、ボールがカーブ
- 各プレイヤー 使用回数は**3回**まで
- ここぞという場面で使うか、温存するかが勝負の鍵

---

## 勝利条件
- **9点先取**で勝利
- 短時間で決着がつくテンポの良い試合展開

---

## 操作方法

### Player Red
| 操作 | Key |
|---|---|
| 上下移動 | W / S |
| 回転 | D / A |
| カーブショット | Space |

### Player Blue
| 操作 | Key |
|---|---|
| 上下移動 | ↑ / ↓ |
| 回転 | → / ← |
| カーブショット | Right Shift |
