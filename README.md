# Mono Rink

## 概要
本作は、ローカル対戦専用の2人用エアホッケーゲームです。  \
シンプルな操作ながら、カーブショットによる駆け引きが特徴です。\
\

ジャンル：対戦型エアホッケーゲーム\
プレイ人数：２人\

## 操作方法

### Player Red
| 操作 | Key |
|---|---|
| 上下移動 | W / S |
| 回転 | D / A |
| カーブショット | Space |

### Player Blue
| 操作 | Key |
|---|---|
| 上下移動 | ↑ / ↓ |
| 回転 | → / ← |
| カーブショット | Right Shift |

## カーブショット

- ボールとプレイヤーが**接触した瞬間**に上 or 下方向を入力していると、ボールが曲がります
- 各プレイヤー **使用回数は3回まで**
- タイミングと方向入力が重要なテクニックです
